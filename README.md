# JDBC (jdbc)
JDBC (Java Database Connectivity) is a Java API that defines how a client may access a database. It provides methods for querying and updating data in a database and is oriented towards relational databases. JDBC is part of the Java Standard Edition platform, providing a standard interface for connecting Java applications to databases.
